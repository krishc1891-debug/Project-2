This program measures a user’s typing speed and accuracy by having them type a sentence displayed on the screen. 
The user selects a difficulty level, and the program presents a corresponding sentence. When the user begins typing, 
the program records the start time and captures the input. When the user presses ENTER, the program records the end time. 
Using these timestamps, it calculates the user’s words-per-minute (WPM) using the standard formula where one word is 
approximated as five characters. The program also calculates accuracy by comparing the typed text to the expected text 
character-by-character.The program allows the user to take multiple typing tests in one session. Each test result 
is stored in a list as a dictionary containing the difficulty level, elapsed time, WPM, and accuracy. After the 
session ends, the program displays a summary of all tests taken. The program also writes the test history to a 
text file so results can be viewed later. This problem requires the use of functions, strings, loops, conditionals, 
and file I/O.
